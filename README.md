## 5.2管理员模块实现
管理员进入主页面，主要功能包括对系统首页、个人中心、员工管理、客户信息管理、设备类型管理、设备型号管理、设备信息管理、设备维修管理、设备保养管理、定检信息管理、配件信息管理、配件采购管理、合同信息管理等进行操作。管理员主页面如图5-3所示：

![](/md/blog.014.png)

图5-3 管理员主界面

![](/md/blog.015.png)

图5-4客户信息管理界面
